
## 使用说明
- 登录角色介绍
   -  博客拥有者 (BeMount) 
   -  普通用户，可自己注册
   -  游客
- 不同权限
   - 拥有者可以编辑文章，发布文章
   - 登录者可以收藏文章，查看文章，对文章评论进行管理
- 技术栈
   - Vue
   - Nodejs
   - Mongodb
   - Nginx
- 功能模块
   - 个人中心 
      - 修改密码
      - 修改头像
      - 完善个人信息
      - 查看收藏
   - 评论管理
      - 查看评论
      - 删除评论
   - 编辑文章
      - 发布文章
      - 使用markdown格式
      - 选择标签
      - 添加标签
      - 添加文章描述
   - 文章管理
      - 查看文章
      - 收藏文章
   - 作者相关
      - 有关作者的静态页面
  
- 密码
  - 博客拥有者密码为123456,请使用BeMount用户名登录。(仅用于测试)
